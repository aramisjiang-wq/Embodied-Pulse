## 1. 产品概述

### 1.1 产品愿景

为Embodied Pulse平台的市场人员提供一个B站UP主（机器人厂家）数据分析页面，展示各UP主的发布量和播放量数据，支持竞品分析、内容运营策略制定和数据汇报。

### 1.2 目标用户

| 用户类型 | 描述 | 使用场景 |
|---------|------|---------|
| 内部市场团队 | 公司内部市场运营人员 | 竞品分析、内容策略制定、汇报材料准备 |
| VIP用户 | 付费升级的注册用户 | 了解行业动态、竞品监控 |

### 1.3 核心价值

1. **竞品分析视角** - 各机器人厂家的内容产出对比，谁发得多、谁播放高
2. **内容运营视角** - 发布频率、发布时间规律，指导内容策略
3. **数据汇报视角** - 可视化数据展示，支持决策和汇报

## 2. 用户故事

### 2.1 市场人员

- 作为市场人员，我想要查看所有关注的UP主列表，以便了解监控范围
- 作为市场人员，我想要查看各UP主的发布量趋势，以便分析竞品内容产出节奏
- 作为市场人员，我想要查看各UP主的播放量数据，以便评估内容影响力
- 作为市场人员，我想要对比多个UP主的数据，以便进行竞品分析
- 作为市场人员，我想要筛选特定时间范围的数据，以便分析特定时期的表现

### 2.2 VIP用户

- 作为VIP用户，我想要访问数据分析页面，以便了解行业动态
- 作为VIP用户，我想要查看UP主排行榜，以便了解行业竞争格局

## 3. 功能需求

### 3.1 筛选控制区

#### 3.1.1 UP主多选
- **优先级**: P0
- **描述**: 支持多选UP主进行数据筛选和对比
- **验收标准**:
  - [ ] 显示所有已添加的UP主列表
  - [ ] 支持搜索UP主名称
  - [ ] 支持全选/取消全选
  - [ ] 已选UP主显示为标签形式

#### 3.1.2 时间范围选择
- **优先级**: P0
- **描述**: 支持选择数据统计的时间范围
- **验收标准**:
  - [ ] 提供快捷选项：近7天、近30天、近90天、近一年
  - [ ] 支持自定义开始和结束日期
  - [ ] 默认选择近30天

#### 3.1.3 数据聚合维度
- **优先级**: P1
- **描述**: 支持按日/周/月聚合数据
- **验收标准**:
  - [ ] 提供日/周/月三个选项
  - [ ] 默认按周聚合
  - [ ] 切换维度后图表自动更新

### 3.2 概览统计卡片

#### 3.2.1 核心指标展示
- **优先级**: P0
- **描述**: 展示选中UP主的核心统计数据
- **验收标准**:
  - [ ] 显示UP主数量
  - [ ] 显示总视频数
  - [ ] 显示总播放量（格式化显示，如1.2M）
  - [ ] 显示本月新增视频数
  - [ ] 显示平均播放量
  - [ ] 显示数据最后更新时间

### 3.3 发布趋势图

#### 3.3.1 多UP主发布趋势
- **优先级**: P0
- **描述**: 展示选中UP主在时间范围内的发布量趋势
- **验收标准**:
  - [ ] 使用折线图展示
  - [ ] X轴为时间，Y轴为发布数量
  - [ ] 不同UP主使用不同颜色区分
  - [ ] 支持鼠标悬停显示详细数据
  - [ ] 支持图例点击隐藏/显示特定UP主

### 3.4 播放量热力图

#### 3.4.1 UP主×月份播放矩阵
- **优先级**: P0
- **描述**: 以热力图形式展示各UP主各月份的播放量
- **验收标准**:
  - [ ] X轴为月份，Y轴为UP主
  - [ ] 颜色深浅表示播放量大小
  - [ ] 支持鼠标悬停显示具体数值
  - [ ] 提供颜色图例说明

### 3.5 UP主排行榜

#### 3.5.1 多维度排行
- **优先级**: P0
- **描述**: 展示UP主排名列表，支持多维度排序
- **验收标准**:
  - [ ] 默认按总播放量排序
  - [ ] 支持按发布数量、平均播放量排序
  - [ ] 显示排名、UP主名称、头像、视频数、总播放量、平均播放、点赞数、收藏数
  - [ ] 点击UP主可展开详情

### 3.6 UP主详情面板

#### 3.6.1 基本信息展示
- **优先级**: P0
- **描述**: 展示单个UP主的详细分析数据
- **验收标准**:
  - [ ] 显示UP主头像、名称、简介、标签
  - [ ] 显示关注时间、最后同步时间
  - [ ] 显示统计概览：总视频数、总播放量、平均播放、点赞数、收藏数

#### 3.6.2 发布规律分析
- **优先级**: P1
- **描述**: 分析UP主的发布习惯
- **验收标准**:
  - [ ] 显示平均发布频率（视频/周）
  - [ ] 显示偏好发布日期（如：周三、周五）
  - [ ] 显示偏好发布时段（如：18:00-20:00）
  - [ ] 显示发布趋势图

#### 3.6.3 播放表现分析
- **优先级**: P1
- **描述**: 分析UP主的播放数据表现
- **验收标准**:
  - [ ] 显示播放量趋势图
  - [ ] 显示点赞率、收藏率
  - [ ] 显示播放量分布（10K以下、10K-50K、50K+）

#### 3.6.4 热门视频TOP10
- **优先级**: P1
- **描述**: 展示UP主播放量最高的10个视频
- **验收标准**:
  - [ ] 显示视频标题、发布日期、播放量、点赞数
  - [ ] 按播放量降序排列
  - [ ] 点击可跳转到B站视频页面

### 3.7 VIP权限控制

#### 3.7.1 页面访问控制
- **优先级**: P0
- **描述**: 仅VIP用户可访问此页面
- **验收标准**:
  - [ ] 非VIP用户访问时显示升级提示
  - [ ] VIP用户可正常访问
  - [ ] 未登录用户跳转到登录页

#### 3.7.2 权限配置功能
- **优先级**: P0
- **描述**: 支持在管理端配置VIP可访问的页面
- **验收标准**:
  - [ ] 管理端可设置用户的VIP状态
  - [ ] 管理端可配置用户的vipPermissions列表
  - [ ] 前端路由守卫检查权限
  - [ ] 后端API验证权限

## 6. 非功能需求

### 6.1 性能要求
- 页面首次加载时间 < 3秒
- 数据查询响应时间 < 2秒
- 图表渲染时间 < 1秒

### 6.2 安全要求
- 页面访问需验证VIP权限
- API请求需验证用户身份和权限
- 敏感数据不暴露给非VIP用户

### 6.3 兼容性要求
- 支持Chrome、Firefox、Safari最新版本
- 支持响应式布局，适配桌面端（1280px+）

## 9. 设计规范

### 9.1 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #1890ff | 品牌蓝色，用于按钮、链接 |
| 辅助色 | #52c41a | 成功/增长，用于正向数据 |
| 警告色 | #faad14 | 警告/注意 |
| 错误色 | #ff4d4f | 错误/下降 |
| 背景色 | #f5f5f5 | 页面背景 |
| 卡片背景 | #ffffff | 卡片/表格背景 |
| 文字主色 | #262626 | 主要文字 |
| 文字次色 | #8c8c8c | 次要文字 |

**禁止使用**：任何紫色作为Banner或按钮背景色

| 元素 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 20px | 600 |
| 卡片标题 | 16px | 500 |

| 正文内容 | 14px | 400 |

| 元素 | 间距 |
|------|------|
| 页面内边距 | 24px |
| 卡片间距 | 16px |
| 卡片内边距 | 16px |

| 图表类型 | 配色方案 |

## 10. 约束与假设

### 10.1 约束条件
- 数据来源于已同步的B站视频数据，非实时数据
- VIP权限系统需先完善配置功能
- 页面仅支持桌面端访问

### 10.2 假设前提
- UP主数据已通过定时任务同步到数据库
- 用户已理解VIP权限的概念
- 市场人员熟悉数据分析的基本概念

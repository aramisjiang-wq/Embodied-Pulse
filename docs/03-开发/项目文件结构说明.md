# Embodied Pulse 项目文件结构说明

**文档版本**: v1.0
**创建日期**: 2026-02-23
**适用范围**: Embodied Pulse 项目

---

## 项目概述

Embodied Pulse 是一个面向具身智能领域的 YouTube 风格信息流平台，提供论文、视频、GitHub 项目、HuggingFace 模型、招聘岗位等多源资源聚合服务。

---

## 根目录文件

```
Embodied Pulse/
├── README.md                 # 项目说明文档，包含项目介绍、安装步骤、技术栈等
├── docker-compose.yml        # Docker 编排配置，定义数据库、Redis、监控等服务
├── .gitignore               # Git 忽略文件配置
├── .prettierrc              # Prettier 代码格式化配置
├── .cursorrules-lite        # Cursor AI 编辑器规则配置
├── .trae-rules-lite         # Trae AI 助手规则配置
```

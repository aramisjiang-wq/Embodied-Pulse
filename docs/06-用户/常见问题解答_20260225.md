## 一、注册与登录

### Q1: 如何注册账号？

**A**: 
1. 点击页面右上角「登录」按钮
2. 在登录页面点击「注册账号」
3. 输入邮箱地址，获取验证码
4. 设置用户名和密码
5. 完成注册

**注意**: 邮箱验证码有效期为10分钟，请及时使用。

### Q2: 忘记密码怎么办？

**A**:
1. 在登录页面点击「忘记密码」
2. 输入注册时使用的邮箱
3. 收取重置密码邮件
4. 点击邮件中的链接设置新密码

**注意**: 密码重置链接有效期为24小时。

### Q3: 登录状态会保持多久？

**A**: 
- Access Token 有效期：7天
- Refresh Token 有效期：30天
- 在有效期内访问网站会自动刷新登录状态

### Q4: 可以使用第三方账号登录吗？

**A**: 
目前暂不支持第三方账号登录（如微信、GitHub等），请使用邮箱注册。

## 二、内容浏览

### Q5: 不登录可以浏览内容吗？

**A**: 
**可以**。Embodied Pulse 采用零门槛访问设计：
- ✅ 浏览所有公开内容（论文、视频、仓库、模型、职位）
- ✅ 使用搜索和筛选功能
- ✅ 查看内容详情页
- ❌ 收藏、评论、发布内容（需要登录）

### Q6: 如何搜索内容？

**A**:
1. 使用顶部搜索框输入关键词
2. 选择搜索类型（全部/论文/视频/仓库/模型/职位）
3. 使用高级筛选按时间、标签等过滤
4. 搜索结果支持按相关度、时间、热度排序

**技巧**: 
- 使用英文关键词搜索论文效果更好
- 多个关键词用空格分隔，表示AND关系

### Q7: 内容来源是什么？

**A**:
| 内容类型 | 数据来源 | 更新频率 |
|----------|----------|----------|
| 论文 | ArXiv | 每小时 |
| 视频 | Bilibili | 每小时 |
| 仓库 | GitHub | 每15分钟 |
| 模型 | HuggingFace | 每6小时 |
| 职位 | 多平台聚合 | 每天 |

### Q8: 如何查看论文的PDF？

**A**:
1. 进入论文详情页
2. 点击「查看PDF」按钮
3. 或点击论文标题直接跳转ArXiv原文

### Q9: 如何找到论文对应的代码？

**A**:
1. 进入论文详情页
2. 查看「代码」标签
3. 如果有代码关联，会显示GitHub仓库链接
4. 点击即可跳转到代码仓库

## 三、收藏与订阅

### Q10: 如何收藏内容？

**A**:
1. 登录账号
2. 在内容卡片或详情页点击「收藏」图标
3. 收藏的内容可在「个人中心 → 收藏」中查看

### Q11: 如何管理收藏？

**A**:
1. 进入「个人中心 → 收藏」
2. 可以按类型筛选收藏内容
3. 支持批量取消收藏
4. 支持搜索收藏内容

### Q12: 什么是关键词订阅？

**A**:
关键词订阅可以让你自动接收感兴趣的内容更新：
1. 创建订阅，设置关键词（如「强化学习」「机器人控制」）
2. 选择内容类型（论文/视频/仓库等）
3. 当有新内容匹配时，系统会通知你

### Q13: 如何创建订阅？

**A**:
1. 登录账号
2. 进入「订阅」页面
3. 点击「新建订阅」
4. 设置订阅名称、关键词、内容类型
5. 保存订阅

### Q14: 订阅通知如何接收？

**A**:
目前支持站内通知：
- 登录后在顶部导航栏查看通知图标
- 点击可查看订阅更新内容

## 四、社区功能

### Q15: 如何发布帖子？

**A**:
1. 登录账号
2. 进入「社区」页面
3. 点击「发布帖子」按钮
4. 填写标题、内容，选择分类
5. 发布

**注意**: 帖子需要遵守社区规范，禁止发布广告、违规内容。

### Q16: 如何评论和互动？

**A**:
1. 登录账号
2. 进入帖子详情页
3. 在评论区输入内容
4. 支持回复他人评论
5. 可以点赞帖子和评论

### Q17: 帖子可以编辑或删除吗？

**A**:
- **编辑**: 进入「我的社区」，找到帖子，点击编辑
- **删除**: 进入「我的社区」，找到帖子，点击删除

**注意**: 删除后无法恢复，请谨慎操作。

## 五、账号设置

### Q18: 如何修改个人信息？

**A**:
1. 登录账号
2. 点击右上角头像 → 「个人中心」
3. 点击「编辑资料」
4. 修改用户名、头像、个人简介等
5. 保存更改

### Q19: 如何修改密码？

**A**:
1. 登录账号
2. 进入「设置」页面
3. 点击「修改密码」
4. 输入当前密码和新密码
5. 保存更改

### Q20: 如何注销账号？

**A**:
目前暂不支持自助注销账号，如需注销请联系管理员：
- 邮箱: support@embodiedpulse.com

## 六、其他问题

### Q21: 网站支持移动端访问吗？

**A**: 
**支持**。网站采用响应式设计，可以在手机、平板等移动设备上正常访问。建议使用以下浏览器：
- iOS Safari
- Android Chrome
- 微信内置浏览器

### Q22: 内容可以下载吗？

**A**:
- **论文**: 点击PDF链接可下载或在线查看
- **视频**: 跳转到B站观看，不支持下载
- **代码/模型**: 跳转到GitHub/HuggingFace下载

### Q23: 如何反馈问题或建议？

**A**:
1. 在社区发布帖子反馈
2. 发送邮件至: support@embodiedpulse.com
3. 在GitHub提交Issue

### Q24: 网站有API吗？

**A**:
目前API仅供内部使用，暂不对外开放。如有合作需求请联系我们。

### Q25: 如何成为内容贡献者？

**A**:
1. 注册并登录账号
2. 在社区发布高质量内容
3. 持续贡献优质帖子、评论
4. 活跃用户有机会成为社区管理员

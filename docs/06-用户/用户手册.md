## 1. 产品简介

### 1.1 产品定位

Embodied Pulse（具身智能脉搏）是一个专注于具身智能领域的内容聚合和社区平台，为您提供：

- 🌊 **信息流浏览** - YouTube风格的信息流体验，轻松发现优质内容
- 📚 **多源资源聚合** - 论文、视频、开源项目、模型、招聘岗位一站式获取
- 💬 **社区互动** - 发布帖子、评论交流、收藏分享
- 👁️ **零门槛访问** - 无需登录即可浏览所有公开内容

### 1.2 目标用户

| 用户类型 | 典型需求 |
|----------|----------|
| 学术研究人员 | 论文检索、代码参考、会议信息 |
| 工业界工程师 | 开源项目、模型库、招聘动态 |
| AI/ML 学生 | 入门教程、学习路径、项目参考 |
| 科技从业者 | 行业趋势、公司动态、应用案例 |

### 1.3 核心特性

| 特性 | 说明 |
|------|------|
| 免登录浏览 | 所有公开内容无需登录即可查看 |
| 多类型内容 | 论文、视频、仓库、模型、职位 |
| 智能推荐 | 基于兴趣的个性化内容推荐 |
| 关键词订阅 | 订阅感兴趣的关键词，获取更新通知 |
| 社区互动 | 发布帖子、评论、点赞、收藏 |

## 2. 快速开始

### 2.1 访问平台

直接访问平台首页即可浏览所有公开内容，无需注册登录。

**访问地址**：
- 生产环境：https://embodiedpulse.com
- 开发环境：http://localhost:3000

### 2.2 注册账号

**步骤**：

1. 点击页面右上角「登录」按钮
2. 在登录页面点击「注册」链接
3. 输入邮箱地址
4. 点击「发送验证码」获取邮箱验证码
5. 输入收到的验证码
6. 设置密码和昵称
7. 点击「注册」完成

**注册成功后**：
- 自动登录并跳转首页
- 可以使用收藏、订阅、发布等功能

### 2.3 登录账号

**步骤**：

1. 点击页面右上角「登录」按钮
2. 输入注册邮箱
3. 输入密码
4. 点击「登录」

**登录状态保持**：
- 登录状态默认保持7天
- 勾选「记住我」可延长至30天

### 2.4 找回密码

**步骤**：

1. 在登录页面点击「忘记密码」
2. 输入注册邮箱
3. 点击「发送重置邮件」
4. 查收邮件并点击重置链接
5. 设置新密码

## 3. 功能指南

### 3.1 首页信息流

#### 3.1.1 信息流浏览

首页展示个性化推荐内容，支持：

| 功能 | 操作 |
|------|------|
| 无限滚动 | 向下滚动自动加载更多内容 |
| 内容筛选 | 点击顶部Tab切换内容类型 |
| 快速搜索 | 使用顶部搜索框搜索关键词 |
| 查看详情 | 点击内容卡片进入详情页 |

#### 3.1.2 内容类型

| Tab | 内容类型 | 说明 |
|-----|----------|------|
| 推荐 | 全部内容 | 综合推荐信息流 |
| 论文 | ArXiv论文 | 学术研究论文 |
| 视频 | B站视频 | 技术讲解视频 |
| 仓库 | GitHub项目 | 开源代码仓库 |
| 模型 | HuggingFace | AI模型资源 |
| 职位 | 招聘信息 | 行业招聘动态 |

### 3.2 分类浏览

#### 3.2.1 论文列表

**访问路径**：`/papers`

**功能**：
- 按分类筛选论文
- 按时间排序
- 关键词搜索
- 查看论文详情和PDF链接

#### 3.2.2 视频列表

**访问路径**：`/videos`

**功能**：
- 浏览技术视频
- 按UP主筛选
- 按播放量/时间排序
- 视频播放和评论

#### 3.2.3 仓库列表

**访问路径**：`/repos`

**功能**：
- 浏览GitHub开源项目
- 按语言筛选
- 按Star数排序
- 查看项目详情和README

#### 3.2.4 模型列表

**访问路径**：`/huggingface`

**功能**：
- 浏览HuggingFace模型
- 按任务类型筛选
- 查看模型详情
- 获取使用说明

#### 3.2.5 职位列表

**访问路径**：`/jobs`

**功能**：
- 浏览招聘信息
- 按地点/类型筛选
- 查看职位详情
- 获取投递链接

### 3.3 搜索功能

**访问路径**：`/search`

**使用方法**：
1. 在页面顶部搜索框输入关键词
2. 点击搜索或按回车
3. 查看搜索结果

**搜索范围**：
- 论文标题和摘要
- 视频标题和描述
- 仓库名称和描述
- 模型名称和描述

### 3.4 社区功能

#### 3.4.1 浏览社区

**访问路径**：`/community`

**功能**：
- 浏览用户发布的帖子
- 按热门/最新排序
- 查看帖子详情和评论

#### 3.4.2 发布帖子

**前置条件**：需要登录

**步骤**：
1. 访问「我的社区」页面 (`/my-community`)
2. 点击「发布帖子」按钮
3. 填写帖子标题
4. 编辑帖子内容（支持Markdown）
5. 选择分享类型（可选）
6. 点击「发布」

**分享类型**：
- 原创内容：直接发布原创文章
- 分享链接：分享外部链接并添加评论

#### 3.4.3 互动功能

| 功能 | 操作 | 说明 |
|------|------|------|
| 点赞 | 点击❤️图标 | 对帖子或评论点赞 |
| 评论 | 输入评论内容 | 支持楼中楼回复 |
| 收藏 | 点击⭐图标 | 收藏到个人收藏夹 |
| 分享 | 点击分享按钮 | 复制链接或分享到社交媒体 |

### 3.5 排行榜

**访问路径**：`/ranking`

**功能**：
- 查看热门内容排行
- 按类型查看排名
- 发现优质内容

## 5. 常见问题

### Q1: 不登录可以使用平台吗？

**可以**。平台支持游客模式，无需登录即可浏览所有公开内容，包括：
- 首页信息流
- 论文、视频、仓库、模型、职位列表
- 内容详情页
- 社区帖子
- 搜索功能

**需要登录的功能**：
- 发布帖子
- 评论互动
- 收藏内容
- 订阅关键词
- 个人中心

### Q2: 验证码收不到怎么办？

**排查步骤**：
1. 检查邮箱地址是否输入正确
2. 查看邮箱的垃圾邮件/促销邮件文件夹
3. 等待1-2分钟后重新发送
4. 确认邮箱未被平台屏蔽

**如仍无法收到**：
- 尝试更换其他邮箱注册
- 联系管理员协助处理

### Q3: 如何修改密码？

**步骤**：
1. 登录账号
2. 访问「个人中心」→「设置」
3. 点击「修改密码」
4. 输入原密码和新密码
5. 点击「确认修改」

**忘记密码**：
- 在登录页点击「忘记密码」
- 通过邮箱重置密码

### Q4: 登录状态能保持多久？

| 情况 | 保持时间 |
|------|----------|
| 默认登录 | 7天 |
| 勾选「记住我」 | 30天 |
| 手动退出 | 立即失效 |

### Q5: 如何取消收藏？

**步骤**：
1. 访问「收藏」页面
2. 找到要取消的内容
3. 点击已收藏的⭐图标
4. 收藏状态取消

**或者在内容详情页**：
- 点击详情页的收藏按钮取消收藏

### Q6: 如何删除发布的帖子？

**步骤**：
1. 访问「我的社区」页面
2. 找到要删除的帖子
3. 点击「删除」按钮
4. 确认删除

**注意**：删除后无法恢复，请谨慎操作。

### Q7: 内容如何推荐？

平台采用智能推荐算法，推荐依据包括：

| 因素 | 权重 |
|------|------|
| 浏览历史 | 高 |
| 收藏记录 | 高 |
| 订阅关键词 | 中 |
| 点赞记录 | 中 |
| 搜索历史 | 低 |

### Q8: 如何反馈问题或建议？

**方式一**：社区发帖
- 访问社区页面
- 发布带有「反馈」标签的帖子

**方式二**：邮件联系
- 发送邮件至：contact@embodiedpulse.com

### Q9: 如何删除账号？

如需删除账号，请通过以下方式联系管理员：
- 发送邮件至：contact@embodiedpulse.com
- 在社区发布删除请求

**注意**：账号删除后，相关数据将无法恢复。

### Q10: 支持哪些浏览器？

**推荐浏览器**：
- Chrome（推荐）
- Firefox
- Safari
- Edge

**不支持**：
- IE浏览器
- 旧版本浏览器

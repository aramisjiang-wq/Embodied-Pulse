## 一、执行摘要

### 1.1 业务概述

**Embodied Pulse（具身智能脉搏）** 是一个专注于具身智能领域的一站式信息聚合平台，为研究者和从业者提供论文、代码、模型、视频和招聘信息的高效获取服务。

### 1.2 核心价值主张

```
┌─────────────────────────────────────────────────────────────────┐
│                        核心价值主张                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   让研究者用10分钟完成原本1小时的信息获取工作                      │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  聚合 · 聚焦 · 效率 · 连接                                │  │
│   │                                                         │  │
│   │  聚合：论文 + 视频 + 代码 + 模型 + 职位                   │  │
│   │  聚焦：专注具身智能领域                                   │  │
│   │  效率：每天节省1小时信息收集时间                          │  │
│   │  连接：研究者与从业者的桥梁                               │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.3 市场机会

| 维度 | 数据 | 说明 |
|------|------|------|
| TAM（总可及市场） | $500万/年 | 全球5万具身智能研究者 × $100/年 |
| SAM（可服务市场） | $50万/年 | 中文用户5000人 × $100/年 |
| SOM（可获得市场） | $5万/年 | 第一年目标500付费用户 |

### 1.4 商业模式选择

**优先级排序**：B端招聘服务 > Freemium订阅 > 数据服务 > 广告

## 附录

### A. 关键指标定义

| 指标 | 定义 | 计算方式 |
|------|------|----------|
| MAU | 月活跃用户数 | 每月至少访问1次的用户 |
| DAU | 日活跃用户数 | 每日至少访问1次的用户 |
| 付费率 | 付费用户占比 | 付费用户数 / 注册用户数 |
| ARPU | 每用户平均收入 | 总收入 / 付费用户数 |
| LTV | 客户终身价值 | ARPU × 平均留存月数 |
| CAC | 客户获取成本 | 获客总成本 / 新用户数 |
| NPS | 净推荐值 | 推荐者% - 贬损者% |

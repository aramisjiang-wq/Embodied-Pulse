## 1. 产品简介

### 1.1 产品定位

Embodied Pulse 是一个专注于具身智能领域的内容聚合和社区平台，为您提供：

- 🌊 **信息流浏览** - YouTube风格的信息流体验，轻松发现优质内容
- 📚 **多源资源聚合** - 论文、视频、开源项目、模型、招聘岗位一站式获取
- 💬 **社区互动** - 发布帖子、评论交流、收藏分享
- 👁️ **零门槛访问** - 无需登录即可浏览所有公开内容

### 1.2 目标用户

| 用户类型 | 典型需求 |
|----------|----------|
| 学术研究人员 | 论文检索、代码参考、会议信息 |
| 工业界工程师 | 开源项目、模型库、招聘动态 |
| AI/ML 学生 | 入门教程、学习路径、项目参考 |
| 科技从业者 | 行业趋势、公司动态、应用案例 |

## 2. 快速开始

### 2.1 访问平台

直接访问平台首页即可浏览所有公开内容，无需注册登录。

### 2.2 注册账号

1. 点击右上角「登录」按钮
2. 选择「注册」选项卡
3. 输入邮箱地址
4. 点击「发送验证码」获取邮箱验证码
5. 输入验证码并设置密码
6. 完成注册

### 2.3 登录账号

1. 点击右上角「登录」按钮
2. 输入邮箱和密码
3. 点击「登录」

### 2.4 找回密码

1. 在登录页面点击「忘记密码」
2. 输入注册邮箱
3. 查收邮件并点击重置链接
4. 设置新密码

## 3. 功能指南

### 3.1 内容浏览

#### 3.1.1 首页信息流

首页展示个性化推荐内容，支持：
- 无限滚动加载
- 内容类型筛选（论文/视频/仓库/模型/新闻/职位）
- 快捷导航Tab切换

#### 3.1.2 分类浏览

| 板块 | 内容 | 入口 |
|------|------|------|
| 论文 | ArXiv学术论文 | `/papers` |
| 视频 | B站技术视频 | `/videos` |
| 仓库 | GitHub开源项目 | `/repos` |
| 模型 | HuggingFace模型 | `/huggingface` |
| 新闻 | 行业资讯 | `/news` |
| 职位 | 招聘信息 | `/jobs` |

### 3.2 社区功能

#### 3.2.1 发布帖子

1. 登录账号
2. 点击导航栏「我的市集」
3. 点击「发布帖子」
4. 选择分享类型或直接发布原创内容
5. 填写标题和内容
6. 点击「发布」

#### 3.2.2 互动功能

| 功能 | 操作 | 说明 |
|------|------|------|
| 点赞 | 点击❤️图标 | 对帖子/评论点赞 |
| 评论 | 输入评论内容 | 支持楼中楼回复 |
| 收藏 | 点击⭐图标 | 收藏到个人收藏夹 |
| 分享 | 点击分享按钮 | 分享到社交媒体 |

### 3.3 个人中心

#### 3.3.1 个人资料

- 访问路径：`/profile`
- 可编辑：昵称、头像、个人简介

#### 3.3.2 收藏管理

- 访问路径：`/favorites`
- 按类型分类管理收藏内容

#### 3.3.3 订阅管理

- 访问路径：`/subscriptions`
- 创建关键词订阅
- 管理订阅列表

## 4. 常见问题

### Q1: 不登录可以使用平台吗？

可以。平台支持游客模式，无需登录即可浏览所有公开内容。登录后可使用发布、评论、收藏等功能。

### Q2: 验证码收不到怎么办？

1. 检查邮箱是否输入正确
2. 查看垃圾邮件文件夹
3. 等待1-2分钟后重新发送
4. 如仍无法收到，请联系管理员

### Q3: 如何修改密码？

登录后访问「个人中心」→「设置」→「修改密码」，输入原密码和新密码即可修改。

### Q4: 如何删除账号？

请联系管理员处理账号删除请求。

### Q5: 内容如何推荐？

平台采用智能推荐算法，根据您的浏览历史、收藏记录和订阅偏好进行个性化推荐。

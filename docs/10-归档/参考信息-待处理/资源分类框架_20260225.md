## 二、详细分类框架

### 1. 模型 (Model)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  一级分类：模型 (Model)                                                      │
│  定义：用于机器人感知、决策、控制的预训练模型                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  二级分类                                                                    │
│  ├── 1.1 VLA模型 (Vision-Language-Action)                                   │
│  │   ├── 说明：视觉-语言-动作模型，端到端生成机器人动作                       │
│  │   └── 典型资源：RT-1, RT-2, OpenVLA, π0, Diffusion-VLA                   │
│  │                                                                         │
│  ├── 1.2 扩散策略 (Diffusion Policy)                                        │
│  │   ├── 说明：基于扩散模型的动作生成策略                                    │
│  │   └── 典型资源：Diffusion Policy, Diffuser, DP3                          │
│  │                                                                         │
│  ├── 1.3 模仿学习 (Imitation Learning)                                      │
│  │   ├── 说明：从专家演示中学习策略                                         │
│  │   └── 典型资源：ACT, ALOHA, Mobile ALOHA                                 │
│  │                                                                         │
│  ├── 1.4 强化学习 (Reinforcement Learning)                                  │
│  │   ├── 说明：通过奖励信号学习最优策略                                     │
│  │   └── 典型资源：PPO, SAC, TD3, Dreamer                                   │
│  │                                                                         │
│  ├── 1.5 世界模型 (World Model)                                             │
│  │   ├── 说明：预测环境动态变化的模型                                       │
│  │   └── 典型资源：Dreamer, IRIS, UniSim                                    │
│  │                                                                         │
│  ├── 1.6 视觉模型 (Vision Model)                                            │
│  │   ├── 说明：机器人视觉感知模型                                           │
│  │   └── 典型资源：DINO, CLIP, SAM, Depth Anything                          │
│  │                                                                         │
│  ├── 1.7 语言模型 (Language Model)                                          │
│  │   ├── 说明：任务理解与规划的语言模型                                     │
│  │   └── 典型资源：GPT-4, LLaMA, Qwen, ChatGLM                              │
│  │                                                                         │
│  └── 1.8 多模态模型 (Multimodal Model)                                      │
│      ├── 说明：融合视觉、语言、触觉等多模态的模型                            │
│      └── 典型资源：PaLM-E, RT-X, Gemini                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
